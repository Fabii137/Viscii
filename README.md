# Viscii
A simple application to view videos in the terminal

## Requirements:
- CMake 3.16
- C++ 20

## Build
1. Create build directory
```bash
mkdir build
cd build
```
2. Build project
```bash
cmake ..
cmake --build .
```

3. Run project
```bash
./bin/viscii <path/to/file>
./bin/viscii // uses default video
```
